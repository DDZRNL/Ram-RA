PROPOSAL NUMBER: S4.06-8187 (For NASA Use Only - Chron: 013812 )
PROPOSAL TITLE:
An Innovative Architecture for Deep Space RF Communication

TECHNICAL ABSTRACT (LIMIT 200 WORDS)
We propose a new ultra-compact system for constructing X- and Ka- band communications systems, greatly reducing size and weight. We further propose a new architecture within this system to allow high transmit power from a fully characterized and reliable solid state source.

POTENTIAL COMMERCIAL APPLICATIONS
Mobile wireless applications have similar requirements to aerospace communications. Compact, low weight, high data rate components are required for mobile broadband communications, including in particular broadband unlicensed wireless LANs at 5 and 60 GHz.
