Coherent Logix, Incorporated (CLX) proposes the = development of=20 a Software Defined Common Processing System (SDCPS) that leverages = the=20 inherent advantages of an integrated parallel processing = architecture.=20 Targeting a specific waveform (TBD) specified by NASA-GRC in terms = of a=20 high level functional model, e.g. Simulink, the study aims to = examine=20 system performance in a reference implementation based on the = HADS2=20 development system configured for this purpose to include the = HyperX=20 signal processor, data converters and General Purpose Processor = (GPP).=20 Waveform implementation will be executed in stages: (i) port an=20 implementation to the signal processing complex consisting of a = single=20 path per functional block (consuming multiple resources depending = on the=20 complexity of each block) (ii) extend the design to account for = N-modular=20 path redundancy aimed at enhancing system reliability (iii) = integrate in=20 the reference platform to enable real-time performance assessment, = e.g.=20 resource utilization, throughput efficiency and emulated radiation = tolerance.
