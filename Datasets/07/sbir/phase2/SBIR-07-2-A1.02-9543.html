TECHNICAL ABSTRACT (Limit 2000 characters, approximately = 200=20 words)
Researchers at the Institute for Human and Machine = Cognition=20 invented OZ, a primary flight display that provides a single, = unified=20 graphic display of critical flight information. OZ virtually = eliminate the=20 need to scan flight instruments and provides superior performance = for=20 flight in low visibility conditions compared to current integrated = avionics displays. eSky licensed OZ and is commercializing it for = general=20 aviation applications. eSky previously demonstrated the = superiority of OZ=20 over current integrated avionics PFDs in high workload = situations.
NASA=20 Langley and the German Aerospace Laboratory invented researchers = H-mode, a=20 collaborative flight control system that shares flight control = between=20 pilot and automation allowing the pilot to remain in-the-loop with = a high=20 level of situational awareness while retaining aeronautical = decision=20 making authority.
In phase 1 eSky designed an integrated = system (H/OZ)=20 that uses OZ as the primary flight display in the H-mode flight = control=20 system (HFCS). H/OZ allows the pilot to maintain superior = situational=20 awareness while remaining engaged in the managing and directing = the=20 flight. eSky incorporated into both HFCS and OZ graphical elements = new to=20 both that support improved functionality.
H/OZ has the = following=20 primary advantages over current integrated avionics = systems:
superior=20 situational awareness; critical navigational information displayed = on the=20 PFD; real-time seamless allocation of control between pilot and=20 automation; Point to Program (P2P) mechanism for visually = selecting=20 airspace elements for guidance; Focus to Display (F2D) mechanism = for=20 displaying on PFD and MFD screens only situationally relevant = airspace=20 information.
eSky will integrate HFCS and OZ into a prototype = H/OZ=20 simulator with new functionality and conduct extensive usability = and=20 performance tests of the H/OZ system using human volunteers.
